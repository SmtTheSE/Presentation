Model Performance Metrics

Performance Evaluation

We evaluated our model using Mean Absolute Error (MAE) to measure the average magnitude of errors in our predictions. A lower MAE indicates better model performance. Our model demonstrates strong performance with a close training and testing MAE, indicating good generalization without overfitting.

Training MAE

0.068252
Mean Absolute Error on Training Set

This value represents how well our model fits the training data. The low MAE indicates our model has learned the patterns in the data effectively.

Testing MAE

0.070266
Mean Absolute Error on Test Set

This value shows how well our model generalizes to unseen data. The small difference from training MAE (only 0.002) indicates minimal overfitting.

Model Type

Random Forest
Ensemble Learning Algorithm

We chose Random Forest for its robustness to outliers, ability to capture non-linear relationships, and interpretability through feature importance.

Feature Importance Analysis

Understanding which features drive the model's predictions is essential for building trust and validating domain expertise. The chart below shows the relative importance of each feature in our model's decision-making process.

Interpretation Guide

Feature importance values represent the percentage contribution of each feature to the model's predictive power. Higher values indicate that the feature has a stronger influence on the model's predictions.

Key Insights from Feature Importance

  • Tax Assessment Value (13.99%) - The strongest predictor, showing that Zillow's existing valuations are highly informative for error correction
  • Property Size Metrics (8.82% for sq footage, 5.82% for lot size) - Confirm that size is a fundamental driver of property value
  • Geographic Factors (8.28% latitude, 6.82% longitude) - Capture neighborhood effects, school districts, and local demand trends
  • Temporal Features - Help model market volatility and seasonal patterns in real estate transactions

Model Architecture & Deployment

Immediate Deployment

The final, trained Random Forest model is ready to be applied to incoming property data to predict the log-error and consequently refine the estimated sale price.

Technical Implementation

We used scikit-learn's RandomForestRegressor with 100 trees and a maximum depth of 10 to balance performance with generalization.
